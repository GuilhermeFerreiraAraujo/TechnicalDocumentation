Flex Flow

Introduction

The flexbox is a CSS3 construct that provides a way to lay out elements that flow. Flow means that the elements will flow from either left to right, also known as horizontal, or up and down, also known as vertical.

Flexbox layout was officially called CSS Flexble Box Layout Module that was made to improve the items align, directionos and order in the container even when they are with dynamic or even unknown size.

The prime characteristic of the flex container is the ability to modify the width or height of its children to fill the available space in the best possible way on different screen sizes.

Flexbox layout algorithm is direction based unlike the block or inline layout which are vertically and horizontally based. This flexbox layout should be used for small application components.

Flex Container

In order to work with flexbox, you need to first create a flexbox container and to set this container to flexbox you need to set the display property to flexbox.

.flexcontainer {
display: flexbox;
}

Flex direction

The flexbox is oriented based on the flex direction. The flex direction is based on the layout axis. If the layout of the flexbox is column, the layout axis is vertical. If the flexbox layout is row, the layout axis is horizontal.

.flexdirection {
flex-direction: row;
}

The next values are available for the flex direction property:

Property Description
column Flows the child elements of the flexbox across the vertical axis top to bottom.
row Flws the child element of the flexbox along the horizontal axis left to right.
column-reverse Renders the child elements along the vertical axis from the reverse end bottom to top.
row-reverse Renders the child elements along the horizontal axis from the reverse end right to left.

Flex pack

Flex-pack as it is in flex-direction, provides direction to flex items.

.flexcontainer {
flex-pack: end;
}

The follow values are available for the flex pack

Property Description
End Renders the child elements from the end in relation to the layout axis set direction
Start Renders the child elements from the start in relation to the layout axis set direction
center renders the child elements centered on the layout axis
distribute Evenly spaces the child elements along the layout axis.

Flex Wrap

Flex-wrap provides the ability to specify what the browser should do in the event that the content within the flexbox exceeds the available space of the flexbox itself. We can specify if the flexbox should wrap or not wrap.

.flexcontainer {
flex-wrap: end;
}

We can use one of the follow options with flex-wrap property:

Property Description
nowrap This property means that the browser should not wrap the content, meaning that it will overflow the content of the page in case there is no space to continue.
wrap Specifies that the content should wrap in case of no space available to continue.
wrap-reverse The same as wrap but cross-start and cross-end are permuted.

Flex Flow

The flex-flow property supports specifying the wrap style as the second parameter: flex-flow: row wrap. It is basically a shorthand for flex-direction and flex wrap.

.flexcontainer {
flex-flow: row wrap;
}
